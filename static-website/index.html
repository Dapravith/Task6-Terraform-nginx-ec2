Terraform

Declares the VPC, security group, and EC2 instance as reproducible, version-controlled code.

AWS EC2

An on-demand virtual server in the cloud, launched automatically from the Terraform plan.

Nginx

A fast, lightweight web server delivering these static files to your browser in milliseconds.

GitHub

The single source of truth — code is cloned straight from the repo during instance boot.

How it gets here

  1. 1

    terraform apply

    Provisions the network and EC2 instance from declarative configuration.

  2. 2

    Boot script runs

    cloud-init installs Nginx and Git, then clones this repository.

  3. 3

    Nginx serves the site

    Files land in the web root and the page goes live — exactly what you're seeing.
